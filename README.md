# davidRack

 Modulo web desarrollado con Rack-WebServer para ejemplificar el consumo de una API con la gema HTTParty de Ruby

## Build Setup

``` bash
# instala las dependencias dependencies
gem install bundler
bundle install

# Correr servidor
rackup config.ru
```
